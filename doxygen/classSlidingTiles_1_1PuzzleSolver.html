void PuzzleSolver::saveSolution ( GameBoard &  gameBoard)

figures out if the moves tree holds a solution and stores it in the supplied gameBoard

Used a breadth-first search to visit all MoveNode nodes. Since each MoveNode has the endGameState we can load that state into a GameBoard and ask the GameBoard if it is solved.

Parameters
nodethe root node from which to start the search
Returns
